Começando (Getting Started)
===========================


A Inpulso, viu a constante necessidade de fazer o mapeamento de objetos, 
campos e até JSON para objetos do Salesforce. Percebendo esta necessidade 
criamos uma biblioteca para auxiliar nessa conversão de dados externos para
objetos do Salesforce.
A biblioteca se resume em três etapas que consistem em:

*	:doc:`Criar o objeto Origem e seus campos através de metadados </Exemplo/comecandoMetadados>`.
*	:doc:`Criar o mapeamento para um objeto do Salesforce </Exemplo/comecandoMapeamento>`.
*	:doc:`Utilizar o mapeamento criado para a conversão dos dados utilizando os métodos da biblioteca </Exemplo/comecandoMetodos>`.
