Origem
======


O metadado **Origem** é responsável por definer uma origem que será utilizado para o mapeamento.

.. Tip:: 

    +----------------+-------------------------------------+--------------------------------------------------------+
    | Name           | Tipo                                | Descrição                                              |
    +================+=====================================+========================================================+
    | Rótulo         | Texto(40)                           | Nome da Instância                                      |
    +----------------+-------------------------------------+--------------------------------------------------------+
    | Nome de Origem | Texto(40)                           | Nome da API                                            |
    +----------------+-------------------------------------+--------------------------------------------------------+
